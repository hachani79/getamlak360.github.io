# Quick Deployment Guide for Amlak 360

## 5-Minute Setup 🚀

### What You Need:
- GitHub account (free at github.com)
- Your `index.html` file (already prepared ✓)

### Steps:

#### 1️⃣ Create GitHub Repo (2 min)
```
Visit: https://github.com/new
Name: amlak360
Visibility: Public
Click: Create Repository
```

#### 2️⃣ Upload Files (1 min)
```
Click: "Add file" → "Upload files"
Select files:
  - index.html
  - README.md
  - .gitignore
  - LICENSE
Click: "Commit changes"
```

#### 3️⃣ Enable GitHub Pages (1 min)
```
Go to: Settings → Pages
Branch: main
Folder: / (root)
Click: Save
```

#### 4️⃣ Visit Your Site (1 min)
```
Wait 2 minutes
Visit: https://<your-username>.github.io/amlak360/
✅ Done!
```

## Troubleshooting

**Q: Site not showing up?**
- A: Wait 2-3 minutes for GitHub Pages to build
- Check Settings → Pages to see build status

**Q: Want custom domain?**
- A: Settings → Pages → Custom domain
- Point your domain DNS to GitHub

**Q: How to update content?**
- A: Edit `index.html` in GitHub
- Commit → Changes live in 1-2 minutes
