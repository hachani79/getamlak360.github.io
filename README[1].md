## 🛠️ How to Deploy on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click **"New"** to create a new repository
3. Name it: `amlak360` (or any name you prefer)
4. Add description: "Smart Estate & Wealth Management Landing Page"
5. Choose **Public** (so it's accessible)
6. Click **"Create repository"**

### Step 2: Upload Files to GitHub
**Option A: Using GitHub Web Interface (Easiest)**
1. In your new repo, click **"Add file"** → **"Upload files"**
2. Upload:
   - `index.html` (your prepared file)
   - `README.md` (from this package)
   - `.gitignore` (from this package)
3. Add commit message: `Initial commit - Amlak 360 landing page`
4. Click **"Commit changes"**

**Option B: Using Git Command Line**
```bash
git clone https://github.com/<your-username>/amlak360.git
cd amlak360
# Copy index.html, README.md, .gitignore to this folder
git add .
git commit -m "Initial commit - Amlak 360 landing page"
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repo **Settings** (top right)
2. Scroll down to **"Pages"** section (left sidebar)
3. Under **"Source"**, select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **"Save"**
5. Wait 1-2 minutes for deployment
6. Your site URL appears: `https://<your-username>.github.io/amlak360/`

### Step 4: Access Your Live Site
Visit: `https://<your-username>.github.io/amlak360/`
